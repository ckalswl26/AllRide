# 내비게이션 참고 자료 반영 메모

업로드된 내비게이션 예제 3종은 모바일 하단 이동 구조를 설계할 때 참고했습니다.

- `custom_bubble_navigation_bar`: 선택된 탭이 위로 떠오르는 버블 인터랙션
- `navigation-stack`: 화면 간 기능 분리와 현재 위치 강조 방식
- `NavigationTabStrip`: 탭 간 일관된 이동 경험

최종 구현은 React 웹앱에 맞게 새로 작성했습니다. 모바일에서는 `추천 / 미션 / 히스토리 / 커뮤니티` 4개 버튼이 하단에 고정되고, 활성 탭 아이콘이 둥근 버블 형태로 떠오릅니다. 데스크톱에서는 동일한 정보 구조를 좌측 사이드 내비게이션으로 변환합니다.
